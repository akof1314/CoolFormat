Options

Preserve CSS

  • Yes

    pc

    /* color */
h1 {
color:red;
}

  • No

    h1 {
color:red;
}

Sort selectors(caution)

  • Yes

    ss

    h2{color:black;}
h1{color:red;}
==>
h1{color:red;}
h2{color:black;}

  • No

    h2{color:black;}
h1{color:red;}
==>
h2{color:black;}
h1{color:red;}

Sort properties

  • Yes

    sp

    h1{font-size:14px;color:red;}
==>
h1{color:red;font-size:14px;}

  • No

    h1{font-size:14px;color:red;}
==>
h1{font-size:14px;color:red;}

Remove last ;

  • Yes

    rl

    h1{color:red}

  • No

    h1{color:red;}

Remove unnecessary backslashes

  • Yes

    rub

    h1{color:blue;}

  • No

    h1{color:red;c\olor:blue;}

Discard invalid properties

  • Yes

    dip

    h1{color:red;}

  • No

    h1{color:red;font-size-adjust:.58;}
